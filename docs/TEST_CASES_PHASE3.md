## 📎 1. 文件附件上傳測試 (15 用例)

### 1.1 上傳功能 (6 用例)

#### TC-ATTACH-001: 上傳文件 - 正常場景
- **模塊**: 附件上傳
- **優先級**: P0
- **前置條件**: 用戶已登錄，有任務訪問權限
- **測試步驟**:
  1. 選擇任務詳情頁
  2. 點擊上傳附件按鈕
  3. 選擇有效文件（PDF < 10MB）
  4. 確認上傳
- **預期結果**: 
  - 文件上傳成功
  - 返回文件信息（id, fileName, fileSize, url）
  - 附件列表顯示新文件
- **自動化**: ✅ 是
